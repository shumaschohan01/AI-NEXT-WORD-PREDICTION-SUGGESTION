## 🚀 Overview

**Next Word Predictor** is an AI-powered text generation app that predicts the next words based on a single input word, similar to **Google Search autocomplete**.  

It’s a portfolio-ready project demonstrating machine learning, tokenization, and interactive web deployment using **TensorFlow** and **Gradio**.

---

Try it live on Hugging Face Spaces:  
👉 (https://huggingface.co/spaces/ShumasChohan/AI_NEXT_WORD_PREDICTION)

> Type a word in the input box and watch the AI generate the next words in real-time.

---

## 💡 Features

- Predict multiple next words from a single seed word.  
- Adjustable number of predicted words via slider.  
- Handles unknown words safely without crashing.  
- Live predictions for Google-style autocomplete.  
- Lightweight and deployable on Hugging Face Spaces.
